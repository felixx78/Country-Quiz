## Run Locally

Clone the project

```bash
  git clone https://github.com/felixx78/Country-Quiz
```

Go to the project directory

```bash
  cd Country-Quiz
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
